# :star2: React shopping cart      

```sh
npm run dev
```

해당 프로젝트는 리액트 쇼핑몰입니다. 사진과 가격 모델로 FRONTROW의 자료들을 사용하였으며, 해당 프로젝트는 개인 프로젝트로 상업적으로 이용되지 않습니다.   

***

- React / React-Redux / Styled Component   
- express.js / mongodb   

*** 

##### Main page
- react-slick을 사용한 fade slider

##### Product page와 상세페이지
- 카테고리, 페이지네이션 기능 구현
- 수량 관련 기능 구현

##### Cart page
- 카트 확인과 삭제 기능
- 총 가격 계산
- 페이팔로 결제 기능

##### History page
- 결제 내역 확인

##### 반응형
- desktop, tablet, mobile
